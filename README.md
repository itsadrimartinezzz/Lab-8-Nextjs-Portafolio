 # Portafolio Web - Next.js

  Portafolio web personal para presentar proyectos, habilidades y mi contacto.

  ## Sitio desplegado en Vercel

  https://lab-8-nextjs-portafolio.vercel.app

  ## Tecnologías

  - **Next.js 16** 
  - **React 18** 
  - **Tailwind CSS** 
  - **Vercel** 

  ## Secciones

  - Inicio — presentación general y tecnologías
  - Sobre mí — aprendizajes, intereses y experiencia
  - Proyectos — trabajos realizados durante el Web
  - Contacto — formulario y enlaces de contacto


  ## Estructura del proyecto

```
portfolio/
├── app/
│   ├── about/
│   │   └── page.js
│   ├── contact/
│   │   └── page.js
│   ├── projects/
│   │   └── page.js
│   ├── globals.css
│   ├── layout.js
│   └── page.js
├── components/
│   ├── Footer.js
│   ├── Navbar.js
│   └── ProjectCard.js
├── data/
│   └── personal-info.js
├── public/
│   └── images/
│       ├── img1.jpg
│       ├── img2.jpg
│       └── img3.jpg
├── next.config.js
├── tailwind.config.js
└── package.json
```

  ## Extra
  Para correrlo localmente se necesita Node.js instalado.

  npm install 
  
  npm run dev

  Luego solo se necesita abrir http://localhost:3000 en el navegador de tu preferencia.
